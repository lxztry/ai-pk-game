# 智能参赛选手

这是一个更高级的智能参赛选手实现，采用了混合策略和多层次的决策逻辑。

## 特性

- **主动攻击策略**: 智能寻找并攻击最近敌人
- **智能躲避**: 主动躲避危险子弹和敌方攻击
- **补给管理**: 血量低时优先寻找补给
- **武器优化**: 根据距离和威胁程度选择合适的武器
- **战术意识**: 综合考虑多种游戏状态做出最优决策

## 使用方法

```python
from participants.smart_agent.agent import SmartAgent

agent = SmartAgent("SmartAI")
```

## 策略详情

1. **紧急处理**: 躲避子弹、低血量补给
2. **攻击优先级**: 优先攻击最近、血量低的敌人
3. **补给管理**: 血量低于50时寻找补给
4. **武器选择**: 根据距离和威胁程度评估武器价值
5. **探索策略**: 智能随机移动避免撞墙
